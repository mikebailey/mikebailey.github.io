The map shows the relative probability that a Facebook user in the selected country has a friendship link to a Facebook user in every other country.

The Social Connectedness Index is computed as the number of Facebook friendship links between two locations divided by the product of their Facebook user populations, then scaled so only relative values matter (Xx higher SCI = X times more likely to be friends compared to the reference).
